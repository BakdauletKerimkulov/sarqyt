# Sarqyt — Roadmap

## MVP Launch (blockers)

- [ ] **Payment system for KZ** — Kaspi Pay or Halyk Bank integration (Stripe doesn't support KZT)
- [ ] **Production Firebase project** — Create new project in asia-south1 region for low latency in Almaty
- [ ] **App Store / Play Store** — Publish both client and business apps

## Post-Launch

- [ ] **Localization** — KZ/RU/EN language support (strings already marked with .hardcoded)
- [ ] **Performance dashboard** — Analytics for business (sales, revenue, popular items)
- [ ] **Financials / payouts** — Track earnings, payout history, commission breakdown
- [ ] **Team management** — Add staff to store, assign roles (operator, employer)
- [ ] **Business verification (KYC)** — Document upload, review process, verification status
- [ ] **CI/CD** — GitHub Actions for build, test, deploy
- [ ] **Offline mode** — Cache offers and orders locally for poor connectivity
- [ ] **Search** — Search offers/stores by name
- [ ] **Notifications settings** — User preferences for push notification types
- [ ] **Order chat** — In-app messaging between customer and store
- [ ] **Multiple items per order** — Cart-like behavior for multiple items from same store
- [ ] **Scheduled offers** — Business sets up recurring offers in advance
- [ ] **Promo codes / discounts** — Marketing tools for stores
